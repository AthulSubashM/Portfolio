Analysis and Optimisation

Optimisation

Load Times

Reducing a webpage’s load time is one of the best ways of improving the overall user experience of the website. Using Google’s PageSpeed Insights tool, I made sure that all the pages of my website load very quickly and have at least a performance score of 90. The assignments which had a lot of image files had a considerably lower performance score than the ones with only a few images. I improved the performance score for those pages by reducing the image sizes from around 500 – 2000 KB each to around 50 – 300 KB. This drastically improved the load times for assignments 1 and 2 without much change is quality.

Responsiveness

The website was designed with responsivity in mind. All the webpages are highly responsive, and the elements adjust automatically according to the screen size. I used a max width of 768px as the breaking point to switch to a mobile layout. In the mobile layout, the top navigation bar becomes vertical and all elements like images that were displayed side-by-side on the desktop layout and displayed vertically.

Navigation

Navigation was another point of focus while building the website. I made sure that all the assignments can accessed from any page with just one click and all other pages can be accessed with no more than two clicks. Reducing the number of clicks to reach the destination makes navigating across the website much easier for the user and is shown to greatly improve the user experience.

Readability

I have used black text on a light grey background, with a high contrast ratio of around 17:1 to improve the readability of the website. The high contrast makes it easier for user with impaired vision to use the website. Moreover, all the headings are of different complementary colours and sizes which further helps navigating the website much easier.

Use Case Analysis

1. A user is someone who has never been to Trent before, is driving here for a meeting, and needs to find a specific classroom. Discuss how this is done without leaving the Trent page.

The user would most likely either click on the menu button and expand the Services & Support option and then go to Conferences or search for the classroom in the search bar. Both these methods would not help the user to locate the classroom. After searching for 15 mins I would say that either it is not possible to locate a classroom thorough the trent.ca website or the website’s developers have done a bad job at making such a useful tool easy to find. The only thing the user could use the website for in this situation is to get Trent University’s contact number and ask for directions.

2. A prospective student needs to know fees for a domestic student enrolled in the MSc Big Data AMOD programme (not thesis based) at Trent university. Start from trentu.ca and never leave the trentu.ca domain.

First click on the menu button and then expand the Academics menu and select Graduate Programs. On the Graduate Studies webpage, click on the Tuition, Awards & Funding button on the right side of the page. Then after scrolling down press the Graduate Student Fees & Tuition button. On this page, the user could find a link saying Fees for Financial Analytics and Big Data Analytics under the Fees for Graduate Programs heading, which would display the required fees.

3. A first-time student wants to find their grade on an assignment (on blackboard).

To find their grades the user should first log into Blackboard and then open the Global Navigation Menu on the top-right part of the page which has the user’s name on it. In the menu the user should select the fourth icon on the left side which says My Grades when hovered over. Here the user could select which course the assignment is from on the left side and view the grades on the right side.

4. Starting with a fresh browser window, how difficult is it to order the web design textbook for this course without using the author’s name from an online retailer

It is very easy to find this book with the author’s name. The first few results that came upon searching for LEARNING WEB DESIGN A Beginner's Guide to HTML, CSS, JavaScript, and Web Graphics 5th edition on Google leads to Amazon from where the right book can be bought.

5. Could you easily order and uber in Peterborough, UK, from here?

Yes, it possible to easily order an uber in Peterborough, UK from here. This can be done by using the Uber website and clicking on the Ride option and selecting the appropriate pickup location and destination. Then user could then just pay for the ride online.

6. Using duckduckgo.com versus google.ca to determine if a political ad or statement is accurate?

I searched “Are liberals communists?” on both Google and DuckDuckGo and got some wildly different results. Google mainly showed results that talked about the difference between liberals and communists and a few results that state that liberals are not communists. Meanwhile on DuckDuckGo, the majority of the results said that liberals are communists with headlines such as “A Liberal, Is A Progressive, Is A Communist”, “Democrat is the New Communist”, “Liberals are Communists”, etc.
